# 🛒 MyShop - E-Commerce Management System

A modern E-Commerce Management System built with **ASP.NET Core MVC (.NET 10)** following **N-Tier Architecture** and Clean Architecture principles.

---

## 🚀 Features

### ✅ Product Management
- Create Product
- Update Product
- Delete Product
- Product Details
- Product Listing
- Product Image Upload
- Category Selection

### ✅ Category Management
- Create Category
- Update Category
- Delete Category
- Category Listing

### ✅ Architecture
- N-Tier Architecture
- Repository Pattern
- Unit of Work
- Dependency Injection
- AutoMapper
- DTO Pattern
- Service Layer

### ✅ Database
- Entity Framework Core
- SQL Server
- Code First
- Migrations

### ✅ Authentication
- ASP.NET Core Identity
- Custom ApplicationUser
